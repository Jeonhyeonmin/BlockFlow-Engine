## 📝 Project Introduction

Unity Entry Platform Clone is a block coding platform inspired by Entry, an educational programming platform. By combining Unity's powerful game development environment with an intuitive block coding system, anyone can easily create and share their own games.

### ✨ Key Features

- **Intuitive Block Coding System**
  - Easy coding environment with drag-and-drop functionality
  - Comprehensive block set for various game logic implementations
  - Real-time preview support

- **Multiplayer Support**
  - BackEnd server integration
  - Real-time multiplayer implementation with Photon networking
  - Easy server setup and management

- **Customization**
  - Freedom to create and import game assets
  - Various game templates provided
  - Extensible block system

## 🚀 Getting Started

### Prerequisites

- Unity 6 (6000.0.28F1) LTS
- BackEnd SDK
- Photon PUN2
- Visual Studio 2019 or higher

### Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/Unity-Entry-Platform-Clone.git
```

2. Open the project in Unity Hub.

3. Configure BackEnd and Photon settings:
   - Create a project in BackEnd console
   - Get App ID from Photon console
   - Input respective keys in project settings

## 🔧 Development Environment Setup

1. **BackEnd Setup**
   - Create new project in BackEnd console
   - Generate and apply server authentication key
   - Configure game server settings

2. **Photon Setup**
   - Import and configure PUN2
   - Register App ID
   - Set server region

## 📝 프로젝트 소개

Unity Entry Platform Clone은 교육용 프로그래밍 플랫폼인 엔트리(Entry)에서 영감을 받아 제작된 블록 코딩 플랫폼입니다. 유니티의 강력한 게임 개발 환경과 직관적인 블록 코딩 시스템을 결합하여, 누구나 쉽게 자신만의 게임을 만들고 공유할 수 있습니다.

### ✨ 주요 기능

- **직관적인 블록 코딩 시스템**
  - 드래그 앤 드롭 방식의 쉬운 코딩 환경
  - 다양한 게임 로직 구현이 가능한 블록 세트
  - 실시간 미리보기 지원

- **멀티플레이어 지원**
  - BackEnd("뒤끝") 서버 연동
  - Photon 네트워킹으로 실시간 멀티플레이 구현
  - 손쉬운 서버 구축 및 관리

- **커스터마이징**
  - 자유로운 게임 에셋 제작 및 import
  - 다양한 게임 템플릿 제공
  - 확장 가능한 블록 시스템
